\doxysubsection{Detailed Description}
Represents a frustum in 3D space. 

The \mbox{\hyperlink{classfrustum_1_1_frustum}{Frustum}} class defines a frustum using a set of planes. It provides functionality to check if points and lines are within the frustum and to filter lines based on their intersection with the frustum.
