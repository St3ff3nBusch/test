\doxysubsection{Detailed Description}
Represents a 3D pose with various attributes such as position, orientation, and bounding box. 

The \mbox{\hyperlink{classikg_1_1_pose}{Pose}} class encapsulates the properties and behaviors of a 3D pose, including its position in space, orientation, bounding box, and other attributes. It provides methods for calculating distances between poses, interpolating between poses, and other utility functions.
