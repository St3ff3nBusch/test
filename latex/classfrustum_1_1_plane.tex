\doxysubsection{Detailed Description}
Represents a plane in 3D space. 

The \mbox{\hyperlink{classfrustum_1_1_plane}{Plane}} class defines a plane using a normal vector and a point on the plane. It provides functionality to calculate the distance from a point to the plane.
